## 🚀 快速开始

### 前置要求
- **Node.js** >= 18（[下载地址](https://nodejs.org/)）
- **Chrome** 或 **Edge** 浏览器

### 启动步骤

```bash
# 1. 克隆项目
git clone https://github.com/T01Studio/ideameow.git
cd ideameow

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
```

打开浏览器访问 **http://localhost:3000/**，你会看到灵感喵的工作界面。

## 🔌 安装浏览器插件

这是灵感喵的「猫爪」，用来从 AI 网站叼回灵感。

### 安装步骤

1. 打开 Chrome 浏览器
2. 地址栏输入 `chrome://extensions/` 回车
3. 开启右上角的 **「开发者模式」** 开关
4. 点击左上角的 **「加载已解压的扩展程序」**
5. 选择项目中的 **`extension/`** 文件夹
6. 完成后，Chrome 工具栏右侧会出现 🐾 猫爪图标

> **验证安装**：回到 `http://localhost:3000/` 刷新页面，画布顶部应显示「灵感收集插件已就绪」的绿色脉冲点。

## 🎣 从 AI 平台采集灵感

灵感喵目前已支持以下平台的自动采集：

| 平台 | 网址 | 卡片颜色 |
|------|------|----------|
| ChatGPT | chatgpt.com | 🟢 绿色 |
| Kimi | kimi.com / kimi.moonshot.cn | 🟠 橙色 |
| 豆包 | doubao.com | 🟢 绿色 |
| DeepSeek | chat.deepseek.com | 🔵 青色 |
| Gemini | gemini.google.com | 🔵 蓝色 |
| Claude | claude.ai | 🟣 紫色 |

### 采集方式

1. 打开任意支持的 AI 平台
2. **用鼠标划词选中**你想要的文本
3. 选中文字后，上方会浮出一个按钮：
   ```
   🐾 喵一下
   ```
4. 点击按钮，文字即被「叼走」——按钮会变成：
   ```
   ✓ 🐾 喵！已叼走！
   ```
5. 回到灵感喵工作台（`http://localhost:3000/`），你会发现一张新卡片出现在画布中！

> 如果使用的是自定义添加的网址，划词后同样会出现浮动按钮。

## 🧶 灵感画布操作

### 画布基本操作

| 操作 | 方式 |
|------|------|
| **平移画布** | 鼠标右键拖拽 / 双指触控板滑动 |
| **缩放画布** | 滚动滚轮 / 双指捏合 |
| **拖拽卡片** | 按住卡片头部的 `⋮⋮` 抓手图标拖动 |
| **自动整理** | 点击「网格智能整理」按钮 |

### 创建自定义便签

点击画布上方的 **「极速便签」** 按钮，输入文字即可手动创建一张卡片。

### 空画布提示

当画布没有卡片时，你会看到一只睡着的小猫 🐱💤，提示你去 AI 网站收集灵感。

## 📝 剧本编辑器写作

左侧是灵感喵的**终稿编辑器**，支持富文本写作：

### 编辑器功能

| 功能 | 操作 |
|------|------|
| 加粗 | 选中文字，点击 **B** 或 `Ctrl+B` |
| 斜体 | 选中文字，点击 *I* 或 `Ctrl+I` |
| 标题 | 点击 H1 / H2 按钮 |
| 引用块 | 点击 `"` 引号按钮 |
| 列表 | 点击有序/无序列表按钮 |
| 插入图片 | 点击 🖼️ 图片按钮，选择本地图片 |
| 撤销/重做 | `Ctrl+Z` / `Ctrl+Y` |

### 从画布获取内容到编辑器

有 **三种方式** 将画布卡片内容传送到编辑器：

#### 方式一：拖拽整张卡片
直接将卡片头部的 `⋮⋮` 手柄拖动到左侧编辑器区域，卡片内容以引用块格式插入。

#### 方式二：划词部分发送
1. 在卡片上**划词选中部分文字**
2. 卡片底部弹出浮动按钮栏：
   - **「发送到编辑器」** — 将选中文字传到编辑器光标位置
   - **「复制」** — 复制选中文字到剪贴板
3. 你也可以直接使用 `Ctrl+C` 复制，再 `Ctrl+V` 粘贴到编辑器

#### 方式三：标记整张卡片
点击卡片右上角的 `✓` 勾号按钮，整张卡片内容自动传到编辑器，并标记为「已成功并入剧本段落」。

### 编辑器宽度调节

- **拖拽分隔条**：编辑器与画布之间有一条分隔线，鼠标按住左右拖动可调节编辑器宽度
- **收起编辑器**：点击编辑器头部的 `⇥` 按钮可完全隐藏编辑器，画布全屏沉浸
- **展开编辑器**：点击左侧窄条上的 `⇤` 按钮恢复编辑器

## 🎴 卡片的高级玩法

### 卡片状态

- **未使用**：卡片正常显示，内容清晰可见
- **已使用**：卡片变暗并显示「已成功并入剧本段落」遮罩，防止重复使用
- **恢复**：点击已用卡片上的「撤回至灵感池」可恢复为未使用状态

### 卡片来源标识

每张卡片的顶部有一条彩色的线，表明它来自哪个 AI 平台：

| 颜色 | 来源 |
|------|------|
| 绿色 | ChatGPT |
| 橙色 | Kimi |
| 绿色 | 豆包 |
| 青色 | DeepSeek |
| 蓝色 | Gemini |
| 紫色 | Claude |
| 灰色 | 自定义便签 / 其他 |

### 删除卡片

点击卡片右上角的 🗑️ 垃圾桶图标即可删除。

## 📦 导出与分享

编辑器头部提供两种导出格式：

### 导出 Markdown（.md）
点击 **「导出 MD」** 按钮，下载 `.md` 文件，适合导入 Notion、Obsidian 等笔记工具。

### 导出 Word（.doc）
点击 **「导出 Word」** 按钮，下载 `.doc` 文件，可用 Microsoft Word 或 WPS 打开编辑。保留标题、引用、列表、图片等格式。

## 🌐 自定义适配网址

灵感喵支持你自行添加任意网址作为灵感来源。

### 通过插件设置页添加

1. 点击 Chrome 工具栏右侧的 🐾 **猫爪图标**
2. 在弹出的设置页最下方输入框中输入网址
3. 点击 **「添加」**
4. 刷新目标网页，划词即可使用

### 网址格式

支持通配符 `*` 和 `?`：

| 示例 | 匹配范围 |
|------|----------|
| `*example.com*` | 匹配该域名下所有页面 |
| `chat.your-ai.com` | 精确匹配特定子域名 |
| `*//*.ai-chat.com/chat*` | 匹配特定路径 |

### 通过 Web App 查看说明

点击灵感喵主界面顶部的 **「网址设置」** 按钮，查看预设平台列表和自定义配置说明。

## ❓ 常见问题

### Q: 插件安装后，AI 网站上没有出现浮动按钮？
1. 确认插件已启用（`chrome://extensions/` 中 IdeaMeow 开关为开）
2. 确认你在支持的平台上（见上表）
3. 尝试刷新页面
4. 如果是自定义网址，检查是否已在插件设置中添加

### Q: 采集的卡片在工作台看不到？
1. 确认工作台 `http://localhost:3000/` 正在运行
2. 刷新工作台页面
3. 检查画布上方插件连接状态是否为「浏览器插件已在线」

### Q: 编辑器保存的内容在哪里？
所有内容保存在浏览器的 IndexedDB 中（本地存储），完全不上传服务器。清除浏览器数据会丢失保存的内容，请定期导出备份。

### Q: 如何备份数据？
定期点击「导出 Markdown」或「导出 Word」将内容保存到本地磁盘。

### Q: 编辑器不能插入图片？
支持通过工具栏图片按钮选择本地图片，图片以 base64 格式嵌入。大型图片建议压缩后再插入。

### Q: 收起编辑器后如何恢复？
点击左侧窄条上的 `⇤` 展开按钮即可。
